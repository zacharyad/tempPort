Nerf Shooting Order Puzzle

Similar to the Color Scan Project, this puzzle is physical and allows customers to solve using a board (image above, in unfinished state) and a nerf toy-gun. Customers had unlocked a nerf toy-gun prior to encountering this board (above) with three uniquely sized circles. If they shoot one of the circles, a grouping of LEDs will light up and cause one LED to stay lit green (counter for correct hits). This repeats with each hit adding to the continually lit LEDs. If they hit all three circles in the wrong order, the LEDs will flash red and a "bad" sounding set of beeps can be heard. When hit in the correct order the puzzle will flash green LEDs, make a nice sounding set of beeps (first couple notes of a Zelda song) and a 5v relay will trigger a 12v lock to release a physical key. They then continue on to the next puzzle with that new key.

Color Scan Hardware and Software

This is a puzzle (hardware and software) I created. Customers scan anything that is either red, green, or blue. The sensor module scans reflected color values and the code anylyses and unlocks an elecrtic lock, revealing a button of corresponding to the color scanned. Once all locks are opened a color sequence is flashed on LEDs and the buttons are used to enter that sequence, in the same way "simon says" is played. This Project illustrates my skillset to solve unique problems using crafted hardware and write code to with users in mind (both customers and also employees). Technically I am creating a problem with this; a problem people pay to solve. In my escape room business I create the product and in the most recent room I used microprocessors and many sensors to craft the physical puzzles.
